controls

in touchscreen,the controls will be,tap once in the direction you want to go in.tap and drag to use selected item (blocks,weapons,etc.) tap the cube/ball to consume something.(health packs,potions,etc.) if you have auto-consume on, it will automatically consume it if you save it,you will have to do it manually. if you put it in the emergency pack,it will compress it into one emergency pack. use it in emergencies and all the stuff you have compressed in will be consumed, all at the same time.
